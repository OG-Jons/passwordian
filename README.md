# passwordian
Ein Password-Manager, welcher von [@Sl1van] (https://github.com/Sl1van) und [@OG-Jons] (https://github.com/OG-Jons) entwickelt wurde.
## Applikation Lokal starten
### Backend

- Dependencies installieren

```
cd backend
npm ci
```

- Starten

```
npm run start
```

### Frontend

- Dependencies installieren

```
cd frontend
npm ci
```

- Starten

```
npm run preview
```
